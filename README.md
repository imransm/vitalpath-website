# VitalPath Website

This repository hosts the public-facing pages for VitalPath iOS app.

## Pages

- [Privacy Policy](privacy.html)
- [Support](support.html)

## Hosting

These pages are hosted on GitHub Pages at:
- Privacy Policy: `https://YOUR_USERNAME.github.io/vitalpath-website/privacy.html`
- Support: `https://YOUR_USERNAME.github.io/vitalpath-website/support.html`

## Setup Instructions

1. Create a new GitHub repository named `vitalpath-website` (or your preferred name)
2. Make it **public**
3. Push these files to the repository
4. Enable GitHub Pages in repository Settings → Pages
5. Select source: Deploy from a branch → main → / (root)
6. Save and wait ~1 minute for pages to be available

## Updating

Simply edit the HTML files and push changes:

```bash
git add privacy.html support.html
git commit -m "Update privacy policy"
git push
```

Changes will be live within 1-2 minutes.
